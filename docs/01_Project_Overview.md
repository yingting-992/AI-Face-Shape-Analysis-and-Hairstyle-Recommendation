# Project Overview

## 1. Project Motivation

本專案的目標是建立一套完整的 **AI 臉型分析與智慧髮型推薦系統**，透過電腦視覺技術分析使用者臉型，並根據臉型特徵與髮型知識庫提供具備推薦理由的髮型建議。

本專案不只著重於臉型分類模型的準確率，而是希望建立一個更完整的 AI 應用流程，讓使用者能從上傳照片、取得臉型分析結果，到理解推薦髮型原因，形成完整且可展示的應用系統。

## 2. Project Positioning

本專案定位為 **AI Computer Vision Application**，而非單純的模型比較研究。

因此，專案重點包含：

* 系統流程完整性
* 模型推論可靠性
* 推薦邏輯可解釋性
* 使用者互動體驗
* GitHub 作品集展示價值

目前使用的 CNN 架構如 ResNet34 屬於成熟模型，因此本專案不將更換模型或微幅調整超參數作為主要成果，而是將重點放在資料品質、推薦系統設計、系統整合與實際應用價值。

## 3. Main Objectives

本專案主要目標如下：

1. 建立完整的臉型辨識流程
2. 整合人臉偵測與前處理流程
3. 建立臉型分類模型與信心值分析
4. 設計具有推薦依據的髮型推薦知識庫
5. 產生可解釋的推薦理由
6. 建立可展示的 Web Demo
7. 完善 GitHub 文件與專案架構

## 4. System Concept

系統整體流程如下：

```text
使用者上傳照片
        |
        v
人臉偵測
        |
        v
臉部校正與前處理
        |
        v
臉型分類模型
        |
        v
信心值分析
        |
        v
髮型推薦引擎
        |
        v
推薦理由產生
        |
        v
結果展示
```

## 5. Expected Application Value

本系統可作為以下應用展示：

* AI Computer Vision 作品集
* 深度學習模型整合展示
* 智慧推薦系統展示
* GitHub 專案代表作
* 研究所推甄與求職作品展示

與一般模型訓練專案相比，本專案更強調從模型到應用的完整流程，讓成果不只是分類結果，而是一套能被使用者理解與互動的 AI 應用系統。

## 6. Current Development Focus

目前專案的開發重點包括：

* 整理既有模型訓練流程
* 建立完整專案文件
* 補足資料集來源與限制說明
* 設計髮型推薦知識庫
* 改善推薦邏輯與推薦理由
* 建立可互動的展示介面

## 7. Limitation Awareness

目前資料集主要來自公開 Face Shape Dataset，且樣本多以歐美臉孔為主。由於亞洲臉孔比例有限，模型對亞洲使用者的泛化能力仍需進一步驗證。

因此，本專案目前不宣稱適用於所有族群，後續將透過建立亞洲測試集、收集少量亞洲驗證資料與補充資料集分析，提升模型可信度與應用可靠性。
